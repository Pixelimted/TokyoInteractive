Terms of Service

Last updated July 2026

Using the bot

Unknown is provided as-is for moderating your Discord server and connecting to your Roblox game. There's no guarantee it will always be online or bug-free, though we try to keep it running.

AI Moderation is a beta feature

AI Moderation flags messages using pattern matching, not a full understanding of context. It will miss things and occasionally flag something harmless. It never takes action on its own — a human moderator reviews every flag.

Prefix commands

If you enable a text command prefix, Unknown will read the content of messages that start with that prefix in order to parse the command. These commands run the exact same logic as their slash command equivalents.

Using the dashboard

By logging in, you agree that any moderation action or setting change you make through the dashboard is your responsibility, the same as if you typed the command in Discord directly.

The Roblox console

Commands run through /ingame or the dashboard console execute directly inside your live Roblox servers. Don't run commands you wouldn't be allowed to run in-game normally. Server owners are responsible for who they give Manage Server permission to.

Changes

These terms may be updated as the bot changes. Continuing to use the bot after changes means you accept the update.

Privacy Policy

Last updated July 2026

What this page covers

This explains what Unknown stores when you use it in your Discord server or log into the dashboard.

What gets stored

  • Moderation cases: who was banned, kicked, muted, warned, or noted, who did it, and why.
  • Server settings you configure, like your mod role, log channels, prefix, and AI Moderation status.
  • Linked Roblox usernames, if you connect a Discord account to one.
  • Which Roblox servers are currently online, so the console feature knows where to send commands.
  • If AI Moderation is turned on, the text of messages that get flagged for review, so a moderator can see what triggered the flag.
  • If a prefix is set, the text of messages that start with that prefix, so the command can be parsed.

What does not get stored

Your Discord login token is never saved on our servers. It stays in your browser only for the length of your session and is used just to check which servers you can manage. Regular messages that aren't prefix commands and don't get flagged by AI Moderation are never stored or logged.

Who can see this data

Only people with Manage Server permission in a given server can view that server's cases and settings through the dashboard. Nobody can see data for a server they don't have permission in.

Questions

If you want data about your server removed, reach out through the Discord server where you found this bot.

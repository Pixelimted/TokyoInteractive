#mod-log
Core Moderation

Every action, logged twice.

A summary line for the overview channel, a full case card for the detailed log. Nothing gets missed, nothing gets buried.

⛔

Bans & Timed Bans

Permanent or scheduled, with optional message history cleanup and full DM notice to the user.

👢

Kicks

Instant removal with reason logging and role-hierarchy checks baked in.

🔇

Mutes

Native timeout-based mutes up to 28 days, with automatic expiry tracked in the background.

⚠️

Warnings

Every warning counts toward auto-escalation — kick at 3, ban at 5, no manual tracking needed.

🗂️

Case History

Pull a user's full record with /cases, or look up any single case by ID.

🔗

Roblox Linking

Attach a Roblox username to any Discord account so every case carries their in-game identity.

Reference

Commands

Every command respects your mod role and Discord permissions together — both need to check out.

/ban
Ban a user, permanently or for a set duration
/unban
Lift a ban using a user ID
/kick
Remove a user from the server
/mute
Timeout a user for a given duration
/unmute
Lift an active timeout early
/warn
Log a warning, counts toward auto-escalation
/unwarn
Remove a warning by case ID
/note
Attach a private mod-only note to a user
/cases
View a user's full moderation history
/case
Look up one case by its ID
/viewlog
See recent moderation activity server-wide
/reason
Edit the reason on an existing case
/roblox
Link, view, or remove a Roblox username
/purge
Bulk delete messages, optionally by user
/slowmode
Set a channel's slowmode delay
/lock & /unlock
Restrict or restore send permissions in a channel
/userinfo
Pull account age, roles, and case count
/ingame
Pick a live server and run a command inside it
/setup
Configure mod role and log channels

The Part Most Bots Don't Do

Run commands where the game actually lives.

/ingame shows every live Roblox server tied to your game, public and private, with real player counts pulled straight from the servers themselves. Pick one, type a command, and it runs through Cmdr in that exact server — no restart, no waiting.

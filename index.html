Everything Built In

Highly configurable features.

Every setting is adjustable from the dashboard or a single Discord command.

⛔

Bans & Timed Bans

Permanent or scheduled, with optional message history cleanup and a DM notice to the user.

🛡️

Auto-Escalation

Set your own thresholds — auto-kick and auto-ban after a configurable number of warnings.

🗂️

Full Case History

Every action becomes a case with an ID, reason, and moderator attached. Nothing gets lost.

🔗

Roblox Account Linking

Attach a Roblox username to any Discord account so every case carries their in-game identity.

AI Moderation Beta

Flags scam links, slurs, and mass-mention spam to a review channel. Never acts on its own.

Commands

Every command respects your mod role and Discord permissions together — both need to check out.

/ban
Ban a user, permanently or for a set duration
/unban
Lift a ban using a user ID
/kick
Remove a user from the server
/mute
Timeout a user for a given duration
/unmute
Lift an active timeout early
/warn
Log a warning, counts toward auto-escalation
/unwarn
Remove a warning by case ID
/note
Attach a private mod-only note to a user
/cases
View a user's full moderation history
/case
Look up one case by its ID
/viewlog
See recent moderation activity server-wide
/reason
Edit the reason on an existing case
/roblox
Link, view, or remove a Roblox username
/purge
Bulk delete messages, optionally by user
/slowmode
Set a channel's slowmode delay
/lock & /unlock
Restrict or restore send permissions in a channel
/userinfo
Pull account age, roles, and case count
/ingame
Pick a live server and run a command inside it
/setup
Configure everything — one command, every option optional
The Part Most Bots Don't Do

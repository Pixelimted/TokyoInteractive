Highly customizable features.

Everything a moderation bot should have, designed to be as seamless and configurable as possible.

⛔

Bans & Timed Bans

Permanent or scheduled, with optional message history cleanup and a custom DM notice sent directly to the user.

🛡️

Auto-Escalation

Set your own thresholds. Automatically kick or ban users after a configurable amount of recorded warnings.

🗂️

Full Case History

Every action becomes a persistent case with an ID, reason, and moderator attached. Absolutely nothing gets lost.

🔗

Roblox Account Linking

Attach a Roblox username to any Discord account so every mod case perfectly reflects their in-game identity.

⌨️

Custom Prefixes

Prefer typing !ban over slash commands? Set any custom prefix you like instantly from the web dashboard.

🎮

Reaches Into Roblox

Run Cmdr commands directly inside your live Roblox servers, public or private, right from a Discord command.

Beta

AI Moderation

Unknown quietly scans messages for scam links, slurs, and mass-mention spam, cleanly flagging anything suspicious to a private review channel. It never takes automated action on its own — ensuring a human always makes the final call.

Complete Control.

Every command naturally respects both your custom mod role and native Discord permissions seamlessly.

/ban
Ban a user, permanently or for a set duration
/unban
Lift a ban using a user ID
/kick
Remove a user from the server
/mute
Timeout a user for a given duration
/unmute
Lift an active timeout early
/warn
Log a warning, counts toward auto-escalation
/unwarn
Remove a warning by case ID
/note
Attach a private mod-only note to a user
/cases
View a user's full moderation history
/case
Look up one specific case by its ID
/viewlog
See recent moderation activity server-wide
/reason
Edit the reason on an existing case
/roblox
Link, view, or remove a Roblox username
/purge
Bulk delete messages, optionally by specific user
/slowmode
Set a channel's slowmode delay instantly
/lock & /unlock
Restrict or restore send permissions in a channel
/userinfo
Pull account age, roles, and case count metrics
/ingame
Pick a live server and run a command inside it
/setup
Configure everything — one simple command

Run commands where the game actually lives.

/ingame shows every live Roblox server tied to your game, public and private, with real player counts pulled straight from the servers. Pick one, type a command, and it runs through Cmdr in that exact server.
